# 2.15.5 — Alcance Turístico y Multi-país

## Descripción
Combinación de idioma + moneda + accesibilidad para que un mismo local atienda turistas y funcione en distintos países, como argumento de venta del producto.

## Dependencias
- `Alcance_MenuIdiomas.md`, `Alcance_MultiMoneda.md`, `Alcance_AltoContraste.md`.

## Contenido
- Perfil de "modo turístico" que prende idiomas y accesibilidad por defecto.
- El dueño elige en qué mercados quiere estar.
- Se vende el mismo software a locales de distintos países.
- Documentado como diferenciador de venta.

## Criterio de validación
- Existe un modo turístico que agrupa idioma/moneda/accesibilidad.
- El dueño elige mercados objetivo.
- El software es vendible multi-país.
