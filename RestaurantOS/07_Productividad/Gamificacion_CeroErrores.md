# 2.18.2 — Reconocimiento de Cero Errores

## Descripción
Distintivo de "cero errores" por turno (sin anulaciones ni devoluciones evitables), visible para el equipo como reconocimiento.

## Dependencias
- `Anulacion` (2.2.3), `Dueno_PlatosDevueltos.md`, `Gamificacion_RankingTiempos.md`.

## Contenido
- Por empleado/turno: conteo de errores evitables.
- Quien llega a cero errores en el turno recibe distintivo.
- El distintivo es visible para el equipo.
- No expone quién tuvo errores, solo el logro.

## Criterio de validación
- Se calcula cero errores por turno/empleado.
- El distintivo es visible para el equipo.
- No se exponen los errores ajenos.
