# 2.16.3 — No Encuesta Larga (límite de alcance)

## Descripción
Definición de límite: el feedback es de dos segundos y operativo. No se usa para juntar emails ni fidelizar; eso es otra área.

## Dependencias
- `Feedback_CaritaCierre.md`, `Fidelizacion_Logistica.md` (límites).

## Contenido
- En alcance: carita + alerta al dueño.
- Fuera de alcance: encuestas, NPS, captura de contacto.
- El dueño decide si activa el feedback.
- Documentado para no desviar la función.

## Criterio de validación
- Queda documentado qué está fuera de alcance.
- El feedback no captura datos de marketing.
- El dueño puede activar/desactivar.
