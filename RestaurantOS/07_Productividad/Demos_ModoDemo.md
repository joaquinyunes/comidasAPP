# 2.22.5 — Modo Demo con Datos Precargados

## Descripción (Función 4/20)
Una versión con datos ficticios precargados (mesas, pedidos, stock) que cualquier dueño puede tocar y probar sin tocar datos reales. Clave para vender el producto digitalmente sin ir a ningún lado.

## Dependencias
- `Personal_Capacitacion.md` (modo práctica), `Demos_SimuladorCarga.md`.

## Contenido
- Entorno demo con datos de ejemplo realistas.
- El dueño toca las 4 vistas sin afectar su operación.
- Reset del demo a estado inicial.
- Accesible por enlace o botón "probar demo".

## Criterio de validación
- El dueño entra a un demo con datos precargados.
- Sus acciones no afectan datos reales.
- El demo se resetea a estado inicial.
